# D9 第一个 Vibe-Coding 项目

**项目：** 灵犀（LingXi）—— AI 情感陪伴 Web 应用

---

## K-S-T-A-R Trace

### K · 知识
- 有灵犀的产品设计思路和完整构想
- 知道需要前端页面 + 后端 API + 对话逻辑
- 刚学会自己接 API 做小产品
- 有 Vibe coding 的基本调试经验

### S · 情境
- 五一假期，有整块时间可以动手
- 参加教育数字人大赛，需要做出一个真实可用的产品
- 工具：Claude / DeepSeek API、GitHub Pages、Cloudflare Workers
- 没有完整的前端开发经验，边做边学

### T · 任务
把灵犀从想法跑成一个真实可访问的 Web 应用。
验收标准：页面能打开，能和 AI 对话，有基本的情感陪伴交互体验。

### A · 行动
- 用 Vibe coding 方式：描述想要的功能 → AI 生成代码 → 自己测试 → 反馈问题 → 继续调试
- 前端部署在 GitHub Pages
- 后端用 Cloudflare Workers 做代理
- 对话接 DeepSeek API
- 一步一步调，遇到问题就重新描述给 AI，让它帮定位

### R · 结果
- 跑通了。页面可以访问，对话功能可用。
- 比预期强——AI 生成的代码质量不错，很多细节自己想不到它帮补上了
- 也没有想象中神奇——调试过程需要耐心，不能一次到位，方向还是要自己把控
- 学到的：Vibe coding 的核心不是"让AI写代码"，而是"我来掌舵，AI来执行"
